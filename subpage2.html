Matrix Notation

True and False Statements

  • If A + B is defined, then BT+ AT is defined | TRUE
  • If AB is defined, then BTAT is also defined | TRUE
  • det(A) = det(A^T) | TRUE
  • If det(A) != 0 then rank(A) < n | TRUE
  • A system of equations written in form Ax = 0 always has a non-trivial solution if rank(A) is less than the number of equations | TRUE
  • AB = BA if A and B are square matrices and they have the same size | FALSE
  • If the columns of matrix A are linearly dependent, then its rows are also linearly dependent | FALSE
  • A system of equations written in form Ax = b always has a trivial solution | FALSE (could have no solution)
